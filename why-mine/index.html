Common questions about our websites web miner



Why do we utilize web mining?


We utilize web mining to keep this website running without ads and without making it a paid service. We utilize a bit of your computers CPU power to mine cryptocurrency (XMR) to allow us to run this website for free.



Does this permanently affect my computer?


No this does not affect your computer in any way! The miner automatically shuts off when you close this tab, or this web browser. It does not download any software onto your computer at all! The web miner runs using pure JavaScript, which does not allow for downloading any software onto your computer directly. The only thing this web miner uses is your CPU (in our testing about 4%), and shuts off whenever you close this tab.
